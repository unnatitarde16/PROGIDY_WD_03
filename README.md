Tic Tac Toe 

A modern, stylish Tic Tac Toe web game with a retro neon arcade theme. Play against a friend or challenge an AI powered by the Minimax algorithm with alpha-beta pruning.

✨ Features
🧑‍🤝‍🧑 Two Game Modes
Player vs Player (PvP)
Player vs AI 🤖

🧠 Smart AI
Uses Minimax Algorithm
Optimized with Alpha-Beta Pruning
Plays optimally (hard to beat!)

🎨 Modern UI
Neon arcade theme
Smooth animations & hover effects
Responsive design

📊 Scoreboard
Tracks wins for X and O
Tracks draws

🔄 Game Controls
Restart game
Undo move
Reset scores

🎉 Extra Effects
Winning highlight animation
Particle celebration on win
Hover preview of moves

🛠️ Technologies Used
HTML5
CSS3 (Custom properties, animations, responsive design)
JavaScript (Vanilla JS)
